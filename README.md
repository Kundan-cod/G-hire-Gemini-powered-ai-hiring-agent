# G-Hire – Multimodal AI Hiring Agent

G-Hire is a **Multimodal AI Agent** built for the Google AI Studio competition. It acts as an autonomous hiring assistant that understands text, voice, and vision to help you find the best freelancers.

**Category**: **Live Agent** (Real-time interaction).

## Core Requirements Satisfied
1. **Gemini AI**: Powered by `gemini-3-flash-preview`.
2. **Google GenAI SDK**: Built using `@google/genai`.
3. **Google Cloud Services**: Uses **Firestore** for data and **Firebase Auth** for users.
4. **Multimodal**: Supports **Text**, **Voice** (Speech-to-Text), and **Vision** (Image Analysis).

## Features
- **Vision Analysis**: Upload design screenshots to extract technical requirements.
- **Voice Interaction**: Speak to the agent in the AI Chat.
- **Smart Matching**: Weighted ranking system for freelancers.
- **AI Outreach**: Personalized messages generated by Gemini.
- **Real-time Dashboard**: Track your projects and responses.



## Deployment
This application is designed for Google Cloud Run. 
- Build: `npm run build`
- Start: `npm start`
